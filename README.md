- 写下了第一个程序，摄氏/华氏温度转换，时间是2017年6月8日凌晨1:04

    
	val = input("请输入带温度表示符号的温度值(例如: 32C): ")
	if val[-1] in ['C','c']:
    	f = 1.8 * float(val[0:-1]) + 32
    	print("转换后的温度为: %.2fF"%f)
	elif val[-1] in ['F','f']:
    	c = (float(val[0:-1])-32)/1.8
    	print("转换后的温度为: %.2fC"%c)
	else:
    	print("输入有误")
